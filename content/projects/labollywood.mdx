This project is a collection of websites built for **La Bollywood**, a professional dance entertainment company based in Los Angeles. The goal is to create a strong online presence that showcases their **performances, services, and brand identity**.

The website allows users to explore the company, learn about their team, view past performances, and understand how to book a show. It focuses on delivering a **clean, engaging, and user-friendly experience** for potential clients.

To support different marketing needs, the platform was implemented across **multiple versions of the website**, including a main system and several landing pages. All versions share a consistent design, responsiveness, and performance.

### Features

- Showcase of **dance services and performance offerings**
- Detailed **about and team sections**
- **Booking information** and service flow for clients
- **Gallery and media content** for past performances
- **Testimonials and client highlights**
- Fully **responsive design** for all devices
- Clean and modern **UI focused on engagement and conversion**


### Website Structure

The project includes different versions of the website to support various use cases:

### Main Website

A full-featured website with multiple pages such as:

- Home
- About
- Team / Performers
- Services
- Booking Details
- On-Screen / Film & TV Appearances
- Gallery
- Blog

### Landing Pages

Simplified versions focused on:

- Quick introduction to the brand
- Service highlights and packages
- Testimonials and client trust
- Fast and focused user experience


###  Built With

- **Laravel** – Backend and main system structure
- **React + Inertia.js** – Dynamic frontend for the main website
- **Next.js** – Landing page implementations
- **Tailwind CSS** – Responsive and modern UI design
